# Guidelines de Diseño - Recetario Premium

## 1. Identidad Visual
- **Paleta de Colores:** Cálida y orgánica. 
  - Crema (#FFFDF9) como fondo principal.
  - Terracota (#A0522D) para acentos fuertes y botones primarios.
  - Naranja suave (#FF8A5B) para estados activos e indicadores.
- **Tipografía:** SF Pro o Inter. Pesos 'Black' para títulos y 'Medium/Bold' para cuerpo.
- **Bordes:** Radio uniforme de 18px para tarjetas y botones (estilo iPhone nativo).

## 2. Experiencia de Usuario (UX)
- **Navegación:** Tab bar inferior con desenfoque (backdrop-blur) de 80%.
- **Feedback:** Uso de 'Sonner' para notificaciones rápidas y 'Motion' para transiciones suaves entre pestañas.
- **Interactividad:** Temporizadores tipo "Dynamic Pill" que no interrumpen la lectura.

## 3. Componente IA (Chef Vision)
- **Estética:** Modo oscuro forzado (#0F172A) para diferenciar la potencia tecnológica de la IA del resto de la app.
- **Flujo:** Input de enlace -> Simulación de análisis con micro-animaciones -> Redirección al formulario manual pre-rellenado.
